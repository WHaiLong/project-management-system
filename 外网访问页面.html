外网访问配置指南

  1. 使用ngrok（推荐）：访问 ngrok.com 下载并安装
  2. 启动ngrok隧道：运行 ngrok http 8081（前端）和 ngrok http 8080（后端）
  3. 获取外网地址：ngrok会提供类似 https://xxx.ngrok.io 的地址
  4. 云服务器部署：将项目部署到阿里云/腾讯云等云服务器
  5. 配置域名：绑定域名并配置SSL证书
