# 项目管理系统 - 外网访问部署指南

## 🌐 外网访问方案

### 方案一：本地端口映射（最简单）
```bash
# 1. 启动本地服务
./一键部署外网.sh

# 2. 使用ngrok进行端口映射
ngrok http 80
ngrok http 8080

# 3. 获得的外网地址：
# 前端：https://xxxx-xxx-xxx-xxx-xxx.ngrok.io
# 后端：https://yyyy-yyy-yyy-yyy-yyy.ngrok.io:8080
```

### 方案二：云服务器部署（推荐）
```bash
# 1. 购买云服务器（阿里云/腾讯云/AWS）
# 推荐配置：2核4G，CentOS 8+

# 2. 上传项目到服务器
scp -r project-management-system/ root@your-server-ip:/opt/

# 3. 在服务器上部署
ssh root@your-server-ip
cd /opt/project-management-system
./一键部署外网.sh

# 4. 配置域名和SSL
# 购买域名并解析到服务器IP
# 申请SSL证书（Let's Encrypt免费）
```

### 方案三：容器平台部署（最专业）
```bash
# 1. 使用Docker Hub
docker login
docker tag project-management-backend:latest your-dockerhub/backend:latest
docker tag project-management-frontend:latest your-dockerhub/frontend:latest
docker push your-dockerhub/backend:latest
docker push your-dockerhub/frontend:latest

# 2. 在Kubernetes部署
kubectl apply -f k8s-deployment.yaml

# 3. 配置Ingress和SSL
```

## 🔧 配置步骤

### 1. 域名配置
```bash
# DNS解析记录
A记录：project.your-company.com -> 服务器IP
CNAME记录：www.project.your-company.com -> project.your-company.com
```

### 2. SSL证书配置
```bash
# 使用Certbot申请免费证书
certbot --nginx -d project.your-company.com -d www.project.your-company.com

# 自动续期
certbot renew --dry-run
```

### 3. 防火墙配置
```bash
# 开放端口
sudo firewall-cmd --permanent --add-port=80/tcp
sudo firewall-cmd --permanent --add-port=443/tcp
sudo firewall-cmd --permanent --add-port=8080/tcp
sudo firewall-cmd --reload
```

## 🔒 安全配置

### 1. 修改默认密码
```bash
# 修改管理员密码
UPDATE sys_user SET password='新加密密码' WHERE user_name='admin';

# 修改数据库密码
ALTER USER 'pm_user'@'%' IDENTIFIED BY '新密码';
```

### 2. 启用HTTPS
```nginx
# Nginx配置强制HTTPS
server {
    listen 80;
    server_name project.your-company.com;
    return 301 https://$server_name$request_uri;
}
```

### 3. 限制访问IP
```nginx
# 只允许特定IP访问管理后台
location /admin {
    allow 192.168.1.0/24;
    allow 10.0.0.0/8;
    deny all;
}
```

## 🆘 故障排除

### 常见问题
1. **无法访问**：检查防火墙和端口
2. **数据库连接失败**：检查MySQL服务状态
3. **静态资源404**：检查Nginx配置
4. **SSL证书错误**：检查证书路径和权限
