## 🔧 管理命令

```bash
ps aux | grep -E "(java|node|ngrok)"

# 查看日志
tail -f backend/backend.log
tail -f frontend/project-management-frontend/frontend.log
tail -f ngrok-frontend.log
tail -f ngrok-backend.log

# 停止服务
pkill -f "spring-boot:run"
pkill -f "npm run serve"
pkill -f "ngrok"

# 重启服务
./重启服务.sh
```

## 🆘 故障排除
1. **无法访问ngrok地址**：检查ngrok是否正常运行
2. **服务未启动**：检查日志文件
3. **数据库连接失败**：确保MySQL服务运行
4. **前端加载慢**：可能是网络延迟，ngrok免费版有限制
