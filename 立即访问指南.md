## 🆘 常见问题解决

### 1. 无法访问localhost:8081
```bash
# 检查前端服务
ps aux | grep "npm run serve"

# 重启前端
cd frontend/project-management-frontend
npm run serve
```

### 2. 无法访问localhost:8080
```bash
# 检查后端服务
ps aux | grep "spring-boot:run"

# 重启后端
cd backend
mvn spring-boot:run
```

### 3. 管理员登录失败
```sql
-- 重置密码（如果需要）
UPDATE sys_user SET password='加密后的密码' WHERE user_name='admin';
```
