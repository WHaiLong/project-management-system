## 🚀 备用方案列表

### 方案A：HTTPS+访问令牌（推荐）
**优点**: 快速、安全、无需SSH配置
**步骤**:
1. **获取GitLab访问令牌**:
   - 登录 https://gitlab.cihon.cn
   - 头像 → Edit profile → Access Tokens
   - 创建新令牌，勾选 `api`, `read_repository`, `write_repository`
   - 复制令牌：`glpat-xxxxxxxxxxxxxxxxxxxx`

2. **使用令牌上传**:
   ```bash
   # 使用紧急令牌脚本
   ./紧急令牌上传.sh <你的令牌>
   
   # 或手动执行
   TOKEN="你的访问令牌"
   git remote set-url gitlab https://oauth2:${TOKEN}@gitlab.cihon.cn/whl_openclaw/project-management-system.git
   git push gitlab main --force
   ```

### 方案B：HTTPS+账号密码
**优点**: 无需额外配置，直接使用账号密码
**步骤**:
