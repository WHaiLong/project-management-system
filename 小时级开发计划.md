### 🚀 立即开始

#### 后端团队
```bash
cd backend/ruoyi
# 1. 安装依赖
mvn clean install
# 2. 启动项目
mvn spring-boot:run
```

#### 前端团队
```bash
cd frontend/project-management-frontend
# 1. 安装依赖
npm install
# 2. 启动项目
npm run serve
```

#### 测试团队
```bash
# 1. 准备测试数据
cd tests
python3 生成测试数据.py
# 2. 执行测试
./执行测试.sh
```

#### 运维团队
```bash
# 1. 准备Docker环境
docker-compose up -d
# 2. 配置监控
```
