# 项目管理系统 - 后端架构

## 📁 目录结构
```
backend/
├── ruoyi/                    # 若依框架核心
│   ├── ruoyi-admin/         # 后台管理模块
│   ├── ruoyi-common/        # 公共模块
│   ├── ruoyi-framework/     # 框架核心
│   ├── ruoyi-generator/     # 代码生成器
│   ├── ruoyi-quartz/        # 定时任务
│   ├── ruoyi-system/        # 系统模块
│   ├── pom.xml             # Maven配置
│   └── README.md           # 说明文档
├── project/                 # 项目管理模块（自定义）
├── task/                    # 任务管理模块（自定义）
├── workhour/                # 工时管理模块（自定义）
└── README.md               # 后端说明（本文件）
```

## 🎯 模块说明

### RuoYi框架模块
1. **ruoyi-admin** - 后台管理主模块
   - 包含Spring Boot启动类
   - 系统入口和主配置

2. **ruoyi-common** - 公共模块
   - 工具类、常量、枚举
   - 通用工具和辅助类

3. **ruoyi-framework** - 框架核心
   - 权限控制、日志管理
   - 核心框架组件

4. **ruoyi-generator** - 代码生成器
   - 自动生成CRUD代码
   - 提高开发效率

5. **ruoyi-quartz** - 定时任务
   - 任务调度和管理
   - 定时执行任务

6. **ruoyi-system** - 系统模块
   - 用户、角色、权限管理
   - 系统基础功能

### 自定义业务模块
1. **project** - 项目管理模块
   - 项目创建、编辑、删除
   - 项目进度跟踪
   - 项目成员管理

2. **task** - 任务管理模块
   - 任务分配和跟踪
   - 任务状态管理
   - 任务优先级设置

3. **workhour** - 工时管理模块
   - 工时记录和统计
   - 工时审核和结算
   - 薪资计算基础

## 🔧 技术栈
- **后端框架**: Spring Boot 2.7 + RuoYi 4.7
- **数据库**: MySQL 8.0 + MyBatis Plus
- **安全框架**: Spring Security + JWT
- **缓存**: Redis
- **任务调度**: Quartz
- **构建工具**: Maven

## 🚀 快速开始

### 环境要求
- JDK 1.8+
- Maven 3.6+
- MySQL 8.0+
- Redis 6.0+

### 启动步骤
1. 导入数据库脚本
2. 修改数据库配置
3. 启动Redis服务
4. 运行主启动类

### 开发指南
1. 在对应模块中开发业务代码
2. 使用代码生成器提高效率
3. 遵循RuoYi开发规范
4. 编写单元测试
