# 项目管理系统 (Project Management System)

## 项目概述
基于若依框架的互联网软件项目研发管理系统，支持20人团队协作，集成Jira双向同步，按小时计费的薪资计算。

## 技术栈
- 后端：Spring Boot + MyBatis + 若依框架
- 前端：Vue.js + Element UI
- 数据库：MySQL 8.0
- 缓存：Redis
- 部署：Docker + 阿里云

## 项目结构
project-management-system/
├── backend/           # 后端服务
├── frontend/          # 前端应用
├── docs/             # 项目文档
├── scripts/          # 部署脚本
└── tests/            # 测试相关
