## 📖 第一阶段：基础巩固（1-2周）

### 1. Git高级操作学习
#### 🎯 学习目标
- 掌握Git子模块管理
- 熟练使用Git恢复操作
- 理解Git分支管理策略

#### 📚 学习内容
1. **Git子模块**
   - `git submodule add/update/init/deinit`
   - 子模块与主项目的协作
   - 子模块问题排查

2. **代码恢复操作**
   - `git checkout <commit> -- <file>`
   - `git restore` 命令使用
   - `git reflog` 查看操作历史

3. **分支管理**
   - `git rebase` 与 `git merge` 区别
   - `git cherry-pick` 应用
   - 冲突解决策略

#### ✅ 实践任务
1. 创建一个包含子模块的测试项目
2. 模拟代码丢失并恢复
3. 练习分支合并和冲突解决

---

### 2. Maven依赖管理
#### 🎯 学习目标
- 理解Maven多模块项目结构
- 掌握依赖版本管理
- 能够排查依赖冲突

#### 📚 学习内容
1. **POM文件结构**
   - `<parent>` 配置
   - `<modules>` 配置
   - `<dependencyManagement>` 作用

2. **依赖管理**
   - 依赖范围（scope）
   - 依赖排除（exclusion）
   - 依赖传递性

3. **问题排查**
   - `mvn dependency:tree` 分析依赖
   - 依赖冲突解决
   - 版本统一管理

#### ✅ 实践任务
1. 分析当前项目的POM文件结构
2. 使用Maven工具分析依赖树
3. 解决一个模拟的依赖冲突问题
